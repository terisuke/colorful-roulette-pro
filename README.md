# 🎡 Colorful Roulette Pro

美しく、高性能なルーレットアプリケーション

## 📖 概要

Colorful Roulette Proは、リアルな物理演算と美しいビジュアルを備えたルーレットアプリケーションです。何を選ぶか迷ったときに、運に任せて決めることができます。

## ✨ 主な機能

- 🎨 **美しいビジュアル**: Canvas APIを使用した滑らかなルーレット描画
- 🎵 **音響効果**: Web Audio APIによる回転音とファンファーレ
- 🎯 **カスタマイズ可能**: 自由に項目を追加・削除
- 📱 **レスポンシブデザイン**: モバイル・デスクトップ両対応
- 🎪 **結果表示**: 当選結果をモーダルで表示
- 🌈 **カラフルな配色**: 10色のプリセットカラーで自動配色

## 🛠️ 技術スタック

- **フレームワーク**: React 19.2.3
- **言語**: TypeScript
- **ビルドツール**: Vite 6.2.0
- **スタイリング**: Tailwind CSS
- **アイコン**: Font Awesome 6.4.0
- **フォント**: Noto Sans JP

## 🚀 セットアップ

### 前提条件

- Node.js (推奨: v18以上)

### インストール

1. 依存関係をインストール:

```bash
npm install
```

2. 開発サーバーを起動:

```bash
npm run dev
```

3. ブラウザで `http://localhost:3000` を開く

### ビルド

本番用ビルドを作成:
```bash
npm run build
```

ビルド結果をプレビュー:
```bash
npm run preview
```

## 🎮 使い方

1. **項目の追加**: 右側のパネルで項目名を入力し、「追加」ボタンをクリック
2. **項目の削除**: 各項目の横にあるゴミ箱アイコンをクリック
3. **ルーレット開始**: 「スタート！」ボタンをクリックしてルーレットを回転
4. **結果確認**: 回転が止まると、当選結果がモーダルで表示されます

## 🎨 カスタマイズ

### デフォルト項目の変更

`App.tsx`の`items`初期状態を編集することで、デフォルトの項目を変更できます。

### カラーパレットの変更

`App.tsx`の`PRESET_COLORS`配列を編集することで、使用する色を変更できます。
